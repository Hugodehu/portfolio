Weather station project

Here is our weather station project, it is a project that concerns the creation of a weather station with an Arduino Uno, we have available, a pressure sensor, humidity, temperature, a light sensor and other components such as a SD card module, an RTC clock.

The goal of the project is to be able to capture all the data and save them on our SD card in a text file in order to analyze it later by converting it into .csv for example.
